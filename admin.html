Agrega una fila por cada indicador. Sube su Excel de totales y asígnale nombre y meta.
Nombre del Indicador Meta (%) ① Excel Totales

3 Procesar y generar datos.json
Verifica que todos los indicadores tengan su Excel cargado, luego genera el archivo.
4 Descargar y subir a GitHub

⏳ Primero procesa los Excel (Paso 3)

Una vez procesado, aquí aparecerá el botón para descargar el datos.json

📋 Instrucciones para subir a GitHub
1
Descarga el archivo datos.json con el botón de arriba
2
Entra a tu repositorio en github.com
3
Haz clic en el archivo datos.json que ya existe → luego en el ícono ✏️ (editar)
4
Borra todo el contenido y pega el nuevo. O usa el botón "Upload files" para reemplazarlo directamente
5
Haz clic en "Commit changes" → el dashboard se actualiza automáticamente en segundos ✅
